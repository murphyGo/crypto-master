# Crypto Master Project 

## Project 개요
이 프로젝트는 자동화된 크립토 트레이딩 어플리케이션이다.
셀프 피드백 루프로 스스로 개선하고 성과를 높이는 방향으로 진화한다.

이 문서를 기반으로 requirement.md를 생성한다.

## 기술 스택
 - Python 3.10+ 사용
 - Claude

## 주요 기능
 - 비트코인 차트 분석: 특정 알트코인을 선정할 수도 있지만, 비트코인 트레이딩을 주로 한다. 여러 차트 분석 기법으로 트레이팅 포인트를 도출한다.
 - 종목 선정: 크립토들의 차트를 여러 차트 분석 기법으로 트레이팅 포인트를 도출
 - 차트 분석 기법: 차트 분석 기법을 사전 정의해 두고 해당 기법으로 분석을 진행한다. 분석 기법은 md 파일을 통한 자연어 프롬프트로 정의할 수도 있고, 코드 스니펫일 수도 있다. 
 - 트레이딩 전략: 도출된 트레이딩 포인트로 손익비를 계산하고 배율, 진입가 및 익절가, 손절가를 설정한다.
 - 트레이딩 모드: 실전 트레이딩, 모의 트레이딩 둘 다 가능하다.
 - 거래소: 다양한 거래소를 지원할 수 있는 구조여야 한다.
 - 자동화된 차트 분석 기법 생성 및 백테스팅: 기존 차트 분석 기법 성과 기반으로 개선 또는 전혀 새로운 아이디어를 자동으로 제시한다. 또는 사용자가 제시한 방법으로도 아이디어를 생성한다. 그리고 자동으로 백테스팅, Claude 기반 자동화된 피드백 루프로 아이디어를 개선, 보완하여 발전시켜 성과를 높인다. 성과가 만족스러울 경우 정식 차트 분석 기법으로 도입한다.    
 - 차트 분석 및 익절가, 손절가 등의 생성은 일부 코드 스니펫으로 대체할 수 있지만, 모든 분석, 트레이딩, 차트 분석 기법 생성은 Claude AI agent가 담당한다.
 - UI Dashboard: 차트 분석 기법과 진행중인 트레이딩, 차트 분석 기법 생성 현황, 자산 및 성과 요약을 한눈에 확인할 수 있는 대시보드

핵심은 모든 과정이 Claude AI agent로 동작하며, 분석, 트레이딩 기법이 자동화된 피드백 루프로 진화해야 한다는 것이다.
