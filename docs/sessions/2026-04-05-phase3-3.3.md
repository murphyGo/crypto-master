## Key Decisions

| Decision | Rationale |
|----------|-----------|
| Use asyncio.create_subprocess_exec | Async subprocess execution for non-blocking I/O |
| Extract JSON from markdown code blocks | Claude often wraps JSON in ```json blocks |
| Default 120s timeout | Balance between allowing complex analysis and avoiding hangs |
| Wrap ClaudeError in StrategyExecutionError | Consistent error handling in strategy layer |
| Import ClaudeCLI inside analyze() | Avoid circular imports and lazy loading |

## Implementation Details

### ClaudeCLI.analyze() Flow
1. Check `is_available()` - raise ClaudeNotFoundError if not found
2. Execute `claude -p "{prompt}"` via asyncio subprocess
3. Wait with timeout (kill process on timeout)
4. Check exit code - raise ClaudeExecutionError on non-zero
5. Parse response - try markdown code block extraction first
6. Return parsed JSON dict

### PromptStrategy.analyze() Flow
1. Validate OHLCV input (minimum 20 candles)
2. Format prompt with `format_prompt()`
3. Call `ClaudeCLI().analyze(prompt)`
4. Convert response dict to AnalysisResult
5. Wrap errors in StrategyExecutionError
