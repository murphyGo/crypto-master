## Classes Created

### Exception Hierarchy
1. **ExchangeError** - Base exception for all exchange errors
2. **ExchangeConnectionError** - Connection failures
3. **ExchangeAPIError** - API errors with optional error code

### BaseExchange Abstract Class
Abstract methods:
- `connect()` / `disconnect()` - Connection lifecycle
- `get_ohlcv()` - Historical candlestick data
- `get_ticker()` - Current price
- `get_balance()` - Account balances
- `create_order()` / `cancel_order()` - Order management
- `get_order()` / `get_open_orders()` - Order queries

Context manager support via `__aenter__` / `__aexit__`

### Factory Functions
- `register_exchange(name)` - Decorator to register exchange implementation
- `create_exchange(name, testnet)` - Create exchange instance from registry
- `get_available_exchanges()` - List registered exchange names
- `get_configured_exchanges()` - List exchanges with valid credentials
