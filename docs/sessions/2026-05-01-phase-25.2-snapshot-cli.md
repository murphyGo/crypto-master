## Goal

Wire the snapshot loader from 25.1 into `scripts/backtest_baselines.py`
so baselines run from a committed snapshot instead of live Binance.
Operator-gated `--refresh-snapshot` is the only path that touches
mainnet.

## CLI Surface (4 new flags)

- `--snapshot [PATH]` — opt-in reproducible mode. Optional path
  argument (default `data/backtest/snapshots`). Routes OHLCV through
  `SnapshotExchange`.
- `--refresh-snapshot` — operator-gated. Fetches fresh OHLCV from
  live Binance and writes via `save_snapshot`. Prints
  `"WARNING: --refresh-snapshot fetches OHLCV from LIVE Binance
  mainnet."` and exits without running the baselines (operator does a
  follow-up `--snapshot` invocation against the freshly-written
  dataset).
- `--max-snapshot-age-days INT` — default 30 (quant-recommended
  active-use window). Env-overridable via
  `ENGINE_BASELINE_MAX_SNAPSHOT_AGE_DAYS`. Snapshots older than this
  fail loud with `RuntimeError`.
- `--snapshot-root PATH` — companion to `--refresh-snapshot` for
  pointing at a non-default write root.

`--snapshot` and `--refresh-snapshot` are **mutually exclusive** via
`argparse.add_mutually_exclusive_group()`. Pinned by
`test_cli_snapshot_and_refresh_are_mutually_exclusive`.

## SnapshotExchange Shape

Free-standing class in `src/backtest/snapshot.py` (NOT a `BaseExchange`
subclass — the regenerator only consumes `connect`/`disconnect`/
`get_ohlcv`, and the snapshot dataset has no notion of tickers /
balances / orders). Follows the `_FakeBinanceExchange` injection
pattern used by the existing smoke test, plugged in via
`run_all(exchange=...)`.

Loaded off disk via `SnapshotExchange.from_directory(root, pairs)`.

## Determinism Contract

`test_cross_operator_determinism_byte_identical` runs `run_all` twice
into separate dirs against the same snapshot, then asserts:
- `summary.json`: byte-identical.
- `result.json`: identical after scrubbing `run_id` (BacktestResult-
  level UUID) and `trade_id` (per-trade UUID); these are intentionally
  non-deterministic operator-trace identifiers, not inputs. Every
  other field — metrics, trade ledger, equity curve, timestamps — is
  byte-identical.

Quant sign-off: scrubbing UUIDs is acceptable for the determinism
claim. The bytes that matter for the gate decision are pinned.
