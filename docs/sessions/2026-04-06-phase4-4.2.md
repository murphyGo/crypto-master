## Key Decisions

| Decision | Rationale |
|----------|-----------|
| Use existing TradeHistoryTracker | NFR-007/008 already implements trade persistence with mode separation |
| Instant market fills | Simplify simulation; no order book needed for paper trading |
| Margin-based balance locking | Realistic simulation of leveraged trading |
| Mode="paper" for all trades | Required by NFR-008 for mode separation |
| Store balances in memory | Simple approach; persistence not required for session-based simulation |
| Unleveraged P&L for balance | Balance changes reflect actual funds movement |

## Implementation Details

### Position Flow
```
Position (from TradingStrategy.create_position())
  ↓
PaperTrader.open_position(position)
  ├─ Calculate margin: notional / leverage
  ├─ Lock margin from balance
  ├─ Record via TradeHistoryTracker.open_trade(mode="paper")
  └─ Return TradeHistory

PaperTrader.check_exit_conditions(trade_id, current_price)
  ├─ Compare price to stop_loss/take_profit
  └─ Return (should_exit, reason)

PaperTrader.close_position(trade_id, exit_price, reason)
  ├─ Calculate P&L using Position.calculate_pnl()
  ├─ Unlock margin + apply P&L to balance
  ├─ Record via TradeHistoryTracker.close_trade()
  └─ Return updated TradeHistory
```

### PaperTrader Methods
| Method | Purpose |
|--------|---------|
| `open_position(position)` | Open trade from Position, lock margin |
| `close_position(trade_id, exit_price, reason)` | Close trade, update balance, record P&L |
| `check_exit_conditions(trade_id, current_price)` | Check if SL/TP hit |
| `get_balance(currency)` | Get balance for currency |
| `get_open_trades()` | Get all open paper trades |
| `update_unrealized_pnl(current_prices)` | Update P&L for all positions |
| `get_total_equity(quote_currency, prices)` | Get total equity including unrealized P&L |
